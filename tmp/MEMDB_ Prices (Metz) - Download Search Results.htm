Downloading the Results


You can download the result of your query in a format that can be easily imported into a spreadsheet application (like Microsoft Excel). To do that, follow the instructions on this page step by step.


Note: To follow the instructions more easily, you might want to print out this page first.


A. Downloading the results

To keep the downloading time reasonable, the results are stored in files of 1000 records. If more than 1000 records were retrieved for your query, they will be stored in several files. You will need to download each of these files separately.

     Procedure to save a file:

  1. Click on the first file of the list to display it on the screen (see list just below under heading "List of result files for your query").
  2. In your Web browser menu, choose "File" --> "Save as".
  3. In the dialog box that appears, choose the folder in which you want to store your results.
  4. Still in the same dialog box, choose a name for the file. The name extension can be anything except "txt" (For instance "ResultChunk1.htm"). If you choose the extension "txt", the file will not be saved in the proper format.
  5. Click Save.
  6. Go back to the list by clicking on the "Back" button of your Web browser.
  7. Repeat steps 1 to 6 for each file listed.

B. Importing the results into Microsoft Excel 97

  1. Start Excel.
  2. In the menu bar, choose "File" --> "Open"
  3. In the dialog box that appears, go to the folder were your files are located.
  4. Choose "file of type:" --> "All files (*.*)"
  5. Select the first of your result files and click "Open", this will start the "Text import wizard" automatically.
  6. In Step 1 of the wizard, select "Original Data Type" --> "Delimited".
  7. In Step 2, select "Delimiters" --> "Tab"
  8. In Step 3, select "Column Data Format" --> "General"
  9. Click "Finish". At this point the data appears in the spreadsheet.
  10. In the menu bar, choose "File" --> "Save as"
  11. In the dialog box that appears, enter the name of the file with the "xls" extension (for instance "ResultChunk1.xls")
  12. Select "Save as type" --> "Microsoft Excel Workbook (*.xls)"
  13. Click "Save". Your Data is now stored in the MS Excel format.
  14. Repeat steps 2 to 13 for each file.
